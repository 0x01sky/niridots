## Applying Configs

**Note:** Before Copying the config, make sure to backup your current config in case the dotfiles had a problem!

- Copying Config Files:

```bash
git clone https://github.com/0x01sky/niridots dotfiles
cd dotfiles/configs
cp -r * ~/.config # or copy or move what you want to .config
```
- Copying the shell scripts and binaries :

```bash
mkdir -p "$HOME"/.local/bin
cp -r dotfiles/local-bins/* "$HOME"/.local/bin/ && chmod +x "$HOME"/.local/bin/*
```
- Logout and Login back and you're done!
